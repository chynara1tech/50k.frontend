0. После того как мы создали тег нажать ENTER
1. Правила табов:
  - Прежде чем писать код либо содержимое нужно убедиться что в начале открытого и закрытого тега есть один TAB
2.Каждый код должен находиться между открытым и закрытым тегом
3. Каждый тег должен начинаться с < и закрываться с >
4. Каждый открытый тег должен быть параллелен закрытому. И наоборот, каждый закрытый тег должен быть параллелен открытому тегу. 
  Например:
  <div>
      Содержимое 
      Содержимое 
      Содержимое 
      Содержимое 
      Содержимое 
      Содержимое  
  </div>
5. Всегда наблюдайте за активной полосой, которая находиться слева о тега.
6. Каждый внутренний тег либосодержимое должна содержать один TAB вправо.
7. Каждый соседний не должен содержать дополнительный TAB вправо.  
8. Стрелочка  вскрытия кода.
     Всегда пробуйте свернуть свой код
     Если не получается свернуть, то структура неправильная

     Правила атрибутов:
1. class=""  
    -Каждый атрибут должен начинаться со свойства CLASS / ID / DATA-USER
    -Далее пишеться равно = без каких либо пробелов
    -Кавычки после равно пишется без пробелов!
    -Если кавычки начинаются с двойной кавычки ", то она обязательно должна заканчиваться с двойной "
    -Если кавычка начинается с одинарной кавычки ', то она должна заканчиватся с одинарнойт '
