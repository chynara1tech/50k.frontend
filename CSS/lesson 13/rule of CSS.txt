1). Правила подключения стилей прямо на странице
   -используется тег style
   <style>
        правила стилей
   </style>
   - Данный кусок кода CSS должен распологаться перед закрывающим тегом <head>
2). Правила написания стилей:
        - Начинается все с того, кому мы должны присвоить стили?
        class
        id
        tag
        attribute
        - Мы должны открыть и закрыть скобку { }
        Цеолстная картина должна выглядеть примерно так:
            class {

            }   

            id {


            } 

            tag {


            }
3). Закрывающая фигурная скобка должна быть параллельна названию класса или любого свойства
